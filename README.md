# Мой первый проект на Python

Этот проект создан для обучения работе с GitHub. Он содержит простую программу на Python, которая демонстрирует базовые функции: приветствие пользователя и вычисление суммы двух чисел.

## Функционал

*   Приветствует пользователя по имени.
*   Запрашивает два числа и выводит их сумму.
*   Обрабатывает ошибки ввода (если введены не числа).

## Как запустить

1.  Убедитесь, что у вас установлен Python.
2.  Скачайте файл `main.py`.
3.  Запустите его в терминале или командной строке:
    ```bash
    python main.py
